# Projeto relacionado a pagina principal

Website estatico em HTML5, CSS3 e JavaScript, criado para hospedagem via GitHub Pages.

## Arquivos principais

- `index.html`: estrutura da pagina.
- `style.css`: estilos responsivos e animacoes.
- `script.js`: variaveis editaveis, menu, dark mode, contato e pontos de integracao futura.
- `isotipo1.png`: isotipo exibido no canto superior direito.
- `modelo.glb`: modelo 3D a ser adicionado posteriormente.

## Links editaveis

Edite os destinos no topo de `script.js`:

```js
const catalogo = "#catalogo";
const documentacao = "#documentacao";
const sobreCriador = "#criador";
const contato = "#contato";
const instagram = "https://link-do-instagram";
```
